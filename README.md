# 日程調整メーカー (Schedule Adjustment Maker)

面倒なビジネスメールなどの「日程調整テキスト」を、カレンダーをポチポチとクリックするだけで直感的に作成できるブラウザ完結型のツールです。

## 🌟 特徴 (Features)

* **直感的な操作:** カレンダーから候補日をクリックし、時間をドロップダウンで選ぶだけでテキストが自動生成されます。
* **柔軟な時間設定:** 30分単位での時間指定に加え、「AM」「PM」といったざっくりとした指定にも対応。
* **フォーマット設定の自動保存:** 前置きテキスト、結びのテキスト、年号の有無、箇条書きのスタイルなどの設定はブラウザ（Local Storage）に自動保存され、次回起動時もそのまま使えます。
* **ワンクリック・コピー:** 完成したテキストはボタン一つでクリップボードにコピーできます。
* **環境構築不要:** 単一の `index.html` ファイルのみで構成されているため、Node.jsやビルドツール（npm, webpack等）は一切不要です。

## 🚀 使い方 (Usage)

### ローカルで使う場合
1. このリポジトリから `index.html` をダウンロードします。
2. ダウンロードした `index.html` をお好みのウェブブラウザ（Chrome, Edge, Safariなど）でダブルクリックして開くだけで動作します。

### ウェブ上で公開する場合 (GitHub Pages)
このリポジトリは GitHub Pages と非常に相性が良いです。
1. リポジトリの **Settings** > **Pages** を開きます。
2. Source を `Deploy from a branch` にし、Branch を `main` (または `master`) に設定して Save を押します。
3. 数分後、発行されたURLにアクセスするとアプリがウェブ上で公開されます。

## 🛠️ 技術スタック (Tech Stack)

ビルドステップを排除し、手軽にカスタマイズできるようCDNを活用しています。

* **React (v18):** UMD版をCDN経由で直接読み込み
* **Tailwind CSS:** CDN経由でスタイルを適用
* **Babel Standalone:** ブラウザ上でJSXを直接コンパイル
* **Lucide React (Icons):** 外部リクエストを減らすため、必要なSVGアイコンのみをインライン化して使用

## 📝 カスタマイズ方法

テキストのデフォルト値や時間の選択肢を変更したい場合は、`index.html` をテキストエディタで開き、以下の箇所を直接編集してください。

* `timeOptions` 配列: 選択できる時間の追加・変更
* `useState` の初期値: 前置き・結びのテキストのデフォルト値
